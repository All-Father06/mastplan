EXPERIMENT 1:
Create a repository in github:

After that open Git Bash:
write the following commands for pull:

mkdir te    (te is repository name)
cd te
echo "#te" >> README.md
cat README.md
git init
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Ghawde/te.git

touch file1.html
touch file2.html
touch file3.html
git add file1.html file2.html file3.html
git commit -m"add three HTML files"
git push origin main

write the following commands for push:

git clone https://github.com/Ghawde/te.git
cd te
git rm file1.html
git rm file2.html
git rm file3.html
git commit-m "Removed files"
git push

Alternatives:
Git Bash in the directory where the html files are present.
